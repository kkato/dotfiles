# nvim-tree.lua

サイドバーにディレクトリ構造をツリー表示するファイルエクスプローラー。

## キーマップ

| キー | 動作 |
|------|------|
| `<Space>e` | ツリーの表示/非表示を切り替え |

## ツリー内の操作

### ファイル操作

| キー | 動作 |
|------|------|
| `<CR>` / `o` | ファイルを開く / ディレクトリを開閉 |
| `<C-v>` | 垂直分割で開く |
| `<C-x>` | 水平分割で開く |
| `<C-t>` | 新しいタブで開く |
| `<Tab>` | プレビュー（カーソルはツリーに残る） |

### ディレクトリ移動

| キー | 動作 |
|------|------|
| `<BS>` | 親ディレクトリを閉じる |
| `-` | 親ディレクトリへ移動 |
| `W` | すべてのディレクトリを折りたたむ |
| `E` | すべてのディレクトリを展開 |

### ファイル管理

| キー | 動作 |
|------|------|
| `a` | 新規ファイル/ディレクトリ作成（末尾に`/`でディレクトリ） |
| `d` | 削除 |
| `r` | リネーム |
| `x` | 切り取り |
| `c` | コピー |
| `p` | 貼り付け |
| `y` | ファイル名をコピー |
| `Y` | 相対パスをコピー |

### 表示

| キー | 動作 |
|------|------|
| `H` | 隠しファイル（dotfiles）の表示/非表示 |
| `I` | .gitignore対象ファイルの表示/非表示 |
| `R` | ツリーを更新 |
| `q` | ツリーを閉じる |

## コマンド

| コマンド | 動作 |
|------|------|
| `:NvimTreeToggle` | ツリーの表示/非表示 |
| `:NvimTreeFocus` | ツリーにフォーカスを移動 |
| `:NvimTreeFindFile` | 現在のファイルをツリー内で表示 |
| `:NvimTreeCollapse` | すべてのディレクトリを折りたたむ |
