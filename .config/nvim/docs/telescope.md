# telescope.nvim

ファジーファインダー。ファイル検索、文字列検索、バッファ切り替えなどを高速に行える。

## キーマップ

| キー | 動作 |
|------|------|
| `<Space>ff` | ファイル名で検索 |
| `<Space>fg` | ファイル内容をgrep検索 |
| `<Space>fb` | 開いているバッファ一覧 |

## Telescope内の操作

| キー | 動作 |
|------|------|
| `<CR>` | 選択したファイルを開く |
| `<C-x>` | 水平分割で開く |
| `<C-v>` | 垂直分割で開く |
| `<C-t>` | タブで開く |
| `<C-n>` / `<C-p>` | 次/前の候補 |
| `<Esc>` | 閉じる |

## コマンド

| コマンド | 動作 |
|------|------|
| `:Telescope help_tags` | ヘルプを検索 |
| `:Telescope keymaps` | キーマップ一覧 |
| `:Telescope commands` | コマンド一覧 |
