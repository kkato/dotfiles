# nvim-treesitter

高精度なシンタックスハイライトを提供するプラグイン。
言語のパーサーで構文解析するため、標準の正規表現ベースより正確な色分けができる。

## 設定済みの言語

lua, javascript, typescript, python, go, json, yaml, markdown

## コマンド

| コマンド | 動作 |
|------|------|
| `:TSInstall {言語}` | パーサーを追加インストール |
| `:TSUpdate` | 全パーサーを更新 |
| `:TSInstallInfo` | インストール済み言語の一覧 |
| `:InspectTree` | 構文木を表示 |
